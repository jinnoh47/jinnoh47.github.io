# Jin Hyeok Noh — Data Scientist

Personal portfolio site built as a single-page HTML resume.

### Live

[jinnoh47.github.io](https://jinnoh47.github.io/)

### Stack

- Vanilla HTML / CSS / JS (no frameworks, no build step)
- Google Fonts (Bebas Neue, Instrument Serif, Geist Mono)
- Responsive (mobile + print ready)

### Features

- Interactive accordion for experience details
- Scroll-reveal animations
- Custom cursor (desktop)
- Scanline overlay effect
- Dark theme with lime accent
